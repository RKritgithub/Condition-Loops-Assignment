Conditional statements allow the program to make decisions based on whether a condition is **true or false**. They are used to execute different code blocks based on conditions. ### **Types of Conditional Statements:** 1. **If Statement** **Syntax:** ```javascript if (condition) { // code to be executed if the condition is true } ``` 2. **If-Else Statement** **Syntax:** ```javascript if (condition) { // code if true } else { // code if false } ``` 3. *Else If Statement* ```javascript if (condition1) { // code if condition1 is true } else if (condition2) { // code if condition2 is true } else { // code if all conditions are false }
